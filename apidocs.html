Make Your Own API

  • 1. Before setup API, you need to install VS Code and node js
  • 2. After install that apps, you need to create account heroku and mongodb
  • 3. Download JSON converter and Rest API from our github
  • 4. Open Rest API folder in VS Code, then go to config file, you will see "MONGODB_URI"
  • 5. Goto MongoDB account, create your cluster
  • 6. Create new project
  • 7. Named your project
  • 8. Click create project then click build a cluster
  • 9. Slect your plan.
  • 10. Go to network access, add IP Address
  • 11. Select allow access from anywhere then confirm
  • 12. Go to Database Access, Add New Database User
  • 13. Click auto generate password and copy
  • 14. After status changed to active then go to cluster
  • 15. Click connect then connect your application
  • 16. Change version to 2.2 then copy
  • 17. We back to VS Code paste link from mongo db to config.js
  • 18. Paste password that we have copy to that link
  • 19. Change database name with santuyapi
  • 20. Now we check mongodb has integrate or not, open new terminal > type "node server.js" if it show "mongoDB connected means it has integrated
  • 20. Upload this project to your github
  • 21. Give reepository name, and select private repository
  • 21. Check your github, if it has upload go to heroku account to create new project
  • 22. Create app in heroku
  • 23. Select Connect to github
  • 24. Search repository that we have make then connect it
  • 25. Click deploy
  • 26. Then open project jsonconverter that we have download, go to vpngate.net, go to csv list
  • 27. Copy csv from vpngate to simple.csv in jsonconverter
  • 28. If you see (*) in above or below of the csv file please delete (*)
  • 29. After delete (*) go to terminal then type "node index.js"
  • 30. Go to output.json then copy that result, we go to mongoDB
  • 31. Click collection, then add my own data
  • 32. Give name on databasename "santuyapi" and collectioname "posts"
  • 33. Select insert document then paste result from output.json to there
  • 34. Change to {}, delete default data then paste json result
  • 35. now we check our API link project from heroku + /api/posts if that show like pics, API has ready to use
  • 36. Last step, open android project go to serverfreefragment line 113, copy your API link there, dont forget to use "/" in the end example https://testingapikuxcv.herokuapp.com/
