Firebase Setup

  • 1. Open Firebase. link
  • 2. Create new project or use your old project.
  • 3. Set your project name.
  • 4. After finish on creating firebase project you will go to firebase project homepage.
  • 5. Select add new appliction.
  • 6. Fill the package name field with your admin package name (example com.goliathus.admin).
  • 7. Click register apps, next, next and then go to console.
  • 8. Do the same step start point 5, but in this step you will add application for your client package name (example com.goliathus.vpn).
  • 9. So now, you will have 2 applications in your firebase project.
  • 10. Go to project settings.
  • 11. Download google service json.
